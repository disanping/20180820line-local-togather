## 如何使用gitbash 或命令行同步GitHub

### 安装Git bash

	# 安装git的最新版本，目前我安装的是2.18的，一路默认下去即可。会在系统右键菜单上生成两个快捷菜单:git GUI 、 git bash.
	# 安装后git 会自动在系统环境变量的path（注意：不是环境变量中）路径中添加：C:\Program Files\Git\cmd
	# 安装后调用gitbash是有图标的，而我的ThinkPad重装后没有系统状态栏图标
